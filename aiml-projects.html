Problem Statements
  • Current mobility systems have gaps in real-time context and hands-free navigation
  • Detection systems fail in extreme conditions
  • AI hallucinations and slow, non-multimodal interfaces reduce trust and situational awareness
Tech Stacks
  • YOLOv8 for object detection
  • LLAMA32 for hazard processing (15 frames/sec)
  • RAG with vector databases for real-time traffic insights
  • Edge AI for low-latency, high-efficiency performance
Work Sample Only
Outcomes
  • Achieved 94.7% detection accuracy with 68.4% CPU usage
  • Reduced false positives by 40% and delivered 92.4% relevant alerts
  • Proven scalable in vehicles, warehouses, and public transportation.
  • 4.3 seconds latency and enhanced real-time adaptability with Edge AI and RAG integration
Solution Approaches
  • MGIIA combines YOLOv8 (computer vision) and LLMs (natural language processing) for real-time, hands-free guidance
  • Uses multi-source inputs (cameras, sensors, vector databases) to deliver alerts via voice, text, and video
  • Modular, open-source design for scalability across various industries (transportation, warehousing, public spaces)
